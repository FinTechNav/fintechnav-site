About This Site

I built this site from scratch using vanilla JavaScript with React components for the interactive timeline. It's hosted on Netlify with CI/CD integration to my GitHub repository. The site includes server-side functions via Netlify Functions for the contact form and is configured as a Progressive Web App (PWA) for offline capabilities and improved performance.

Tech stack: HTML5, CSS3, JavaScript, React, Netlify Functions, Zoho Mail API for emails, Hotjar for user experience analytics, Google Analytics, and ESLint/Prettier for code quality. Domain registered with OpenSRS.
